Fund • Live on Signet test network only

Fund a session with Signet sats.

Create a fresh ArkAPI session in the browser, choose 500, 1000, or 1500 sats, and pay through either Lightning or Ark. Live on Signet test network only, not Bitcoin mainnet.

What this page does

  • Creates a new session: each payment request returns a new bearer token and funding details.
  • Shows both rails: scan the Lightning invoice or the Ark address QR code from the GUI.
  • Polls activation: once payment lands on Signet, the session status flips to active.

Select a Signet funding amount

Choose one of the preset session amounts below. ArkAPI will create a new Signet payment request and show both payment methods immediately. Live on Signet test network only.

Need Signet sats first? Get them from the Second Signet faucet before creating a payment request here.
