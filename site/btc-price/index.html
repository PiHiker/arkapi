Fund
Bitcoin Price API • Live on Signet test network only

Get the Bitcoin price in one call.

Query the live Bitcoin spot price in USD by default or request a subset of 10 major fiat currencies. Built for dashboards, agents, bots, and lightweight BTC price checks.

1 sat / call
GET /api/btc-price
Defaults to USD

What It Does

ArkAPI Bitcoin Price returns the live Bitcoin spot price rounded to the nearest cent. By default it returns USD, but you can request one currency or a subset of supported fiat codes through query parameters.

  • Default response returns btc_usd
  • Optional single-currency filter with ?currency=CAD
  • Optional subset filter with ?currencies=USD,EUR,CAD
  • Supported set includes USD, EUR, GBP, CAD, JPY, AUD, CHF, CNY, HKD, and SGD

Behavior

This endpoint is intentionally lightweight. It uses GET, has no request body, and is designed for quick market context instead of historical charting or exchange-specific order data.

  • If no filter is passed, the response defaults to USD
  • Single-currency responses are compact and easy to consume in scripts
  • Subset responses work well for dashboards that need only a few fiat pairs
  • Values are rounded to the nearest cent for readability

Example Request

curl -H "Authorization: Bearer ak_your_token" "https://arkapi.dev/api/btc-price?currencies=USD,EUR,CAD" | jq

Use a funded token from the Fund page. No JSON body is required, and omitting query parameters returns USD by default.

Example Response

{ "success": true, "data": { "btc_usd": "68406.28", "btc_eur": "59345.62", "btc_cad": "94738.48", "updated_at": "2026-03-26T19:15:36Z" }, "cost_sats": 1, "balance_remaining": 8637, "response_ms": 4, "endpoint": "/api/btc-price" }

Good Uses

  • Agent decisions that need a quick BTC/USD reference
  • CLI checks before pricing or settlement logic
  • Lightweight BTC widgets and dashboards
  • Market-context enrichment for Bitcoin workflows

For broader platform usage, see the main docs and the live catalog.

Useful Query Patterns

  • /api/btc-price for the default USD price
  • /api/btc-price?currency=CAD for one fiat rate
  • /api/btc-price?currencies=USD,EUR,CAD for a compact multi-currency response
