Privacy Policy • Last updated March 2026

Privacy Policy

ArkAPI is designed for privacy. No accounts, no email addresses, no personal information required.

Overview

ArkAPI provides anonymous, pay-per-call API access. Sessions are created with a token and funded with Bitcoin. We collect the minimum data needed to operate the service.

What we collect

  • Session tokens (randomly generated, not linked to any identity)
  • API call logs (endpoint, timestamp, cost — for billing and rate limiting)
  • IP addresses in standard web server logs (rotated regularly)

What we don't collect

  • Names, email addresses, or contact information (unless you email us)
  • Payment details beyond what's visible on the Bitcoin/Ark network
  • Browser fingerprints or tracking cookies
  • We don't use analytics scripts or third-party trackers

Data retention

Session data is retained for the lifetime of the session (configurable TTL, typically 24 hours of inactivity). Server logs are rotated on a standard schedule. There are no user accounts to delete.

Third-party services

API endpoints may call external services to fulfill requests (e.g., DNS lookups, WHOIS queries). These requests are made server-side by ArkAPI — your IP is not exposed to these services.

Security

All traffic is encrypted via TLS. Session tokens should be treated as secrets. Report security issues to support@arkapi.dev.

Changes

We may update this policy. Changes will be reflected in the "last updated" date above.

Contact

Questions about privacy: support@arkapi.dev
