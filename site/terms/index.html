Overview

ArkAPI provides pay-per-call API access funded with Bitcoin via Ark and Lightning. No accounts are required. You create a session, fund it, and make API calls.

Sessions and payments

  • Sessions are created by generating a token and funding it with a Bitcoin payment
  • Balances are non-refundable once a session is funded
  • Session balances expire after a period of inactivity
  • Pricing is displayed in the API catalog and on the website
